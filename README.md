# real-estate-dashboard
React-based Real Estate Analytics Dashboard with case studies visualization using Recharts and Lucide React

## Overview

This is a comprehensive Real Estate Analytics Dashboard built with React, featuring interactive data visualizations and case study showcases. The dashboard demonstrates real estate performance metrics with beautiful charts powered by Recharts and modern UI components from Lucide React.

## Features

- **Interactive Dashboard**: Visualize real estate case studies with dynamic charts and metrics
- **Case Study Management**: Showcase multiple real estate projects with detailed performance data
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and professional interface with smooth transitions
- **Data Visualization**: Interactive charts including bar charts, line charts, pie charts, and area charts

## Technologies Used

- **React 18**: Modern React with hooks
- **Recharts**: Composable charting library for React
- **Lucide React**: Beautiful and consistent icon set
- **React Scripts**: Build tooling and development server

## Installation

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Setup

1. Clone the repository:
```bash
git clone https://github.com/Khanhere1/real-estate-dashboard.git
cd real-estate-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

### `npm start`

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder. It correctly bundles React in production mode and optimizes the build for the best performance.
